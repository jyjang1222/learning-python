# 파이썬 일차배열
## 기억사항
1. remove메서드와 예악어del을 사용해서 리스트에 값을 삭제할 수 있다.  

a = [10, 20, 30, 40]  

* remove(삭제할 값)를 통해 제거가 가능하다.  
a.remove(30)  

* 아래와 같이 삭제할 수도 있다. (10 삭제)  
a.remove(a[0])  

* del 삭제할값 을 통해 제거가 가능하다.  
del a[0]  

2. len() 함수를 사용해 배열의 길이를 구할 수 있다.  

3. append() 함수를 사용해서 리스트에 값을 추가할 수 있다.
