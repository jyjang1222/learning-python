# 파이썬 함수 문제풀이

## 함수

### 함수의 종류

1. 내장 함수
   - len(), print(), append(), ...
2. 사용자 정의 함수
   - def name():

```python
def sum(a, b):
    return a + b
sum(1, 2)
```

### 함수의 매개변수와 인자

1. 함수의 매개변수와 인자
   1. 함수를 정의할 때 매개변수를 설정할 수 있다.
      - def plus(a , b):
   2. 매개변수 명은 임의로 설정할 수 있다.
   3. 인자의 개수와 매개변수의 개수는 일치해야 한다.
